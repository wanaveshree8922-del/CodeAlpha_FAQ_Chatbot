# CodeAlpha_FAQ_Chatbot

**FAQ Chatbot** is an offline Flutter application developed as part of my **CodeAlpha Flutter Development Internship**.

The application is designed for **Brew & Bliss Café**, allowing customers to quickly find answers to frequently asked questions about the café, menu, pricing, timings, reservations, facilities, and more through a clean chat interface.

---

# ✨ Features

- **Offline FAQ chatbot** (No Internet Required)
- **Keyword & Intent-based FAQ matching**
- **Realistic typing animation**
- **Bot avatar for every chatbot response**
- **Copy message functionality**
- **Premium café-inspired UI**
- **Custom splash screen**
- **Custom app launcher icon**
- **Responsive chat interface**
- **Long-press to copy chatbot messages**
- **Graceful fallback response for unknown questions**

# ⚙️ How It Works

1. User types a question related to Brew & Bliss Café.
2. The chatbot searches the local FAQ dataset.
3. It matches the closest intent using keyword matching.
4. A typing animation is shown before the reply.
5. The chatbot displays the best matching answer.
6. If no suitable answer exists, a friendly fallback response is displayed.

# 🚀 Setup & Installation

### 1. Clone the repository

```bash
git clone https://github.com/wanaveshree8922-del/CodeAlpha_FAQ_Chatbot.git
cd CodeAlpha_FAQ_Chatbot
```

### 2. Install dependencies

```bash
flutter pub get
```

### 3. Run the application

```bash
flutter run
```
